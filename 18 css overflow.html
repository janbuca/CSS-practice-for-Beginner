For example, if you are writing an XHTML document that contains product information, you might want to call it products.html. 4.1 Computers called web servers running specialized software store XHTML documents. Clients (e.g., web browsers) request specific resources such as the XHTML documents from web servers. For example, typing www.deitel.com/books/downloads.html into a web browser’s address field requests downloads.html from the books directory on the web server running at www.deitel.com.We discuss web servers in detail in Chapter 21. XHTML document creators insert comments to improve markup readability and describe the content of a document.
